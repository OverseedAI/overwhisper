Features

Dictation that fits into the tools you already use.

Overwhisper is not a separate notes silo. It is a fast input method for your Mac, with the controls serious daily use needs.

Hotkeys

Toggle or push-to-talk

Use Option+Space to toggle by default, or Option+Shift+Space for push-to-talk. Both shortcuts are configurable.

Output

Paste at the cursor

Dictate into Messages, Slack, Notes, browsers, editors, issue trackers, or anywhere else macOS accepts text.

Recording

Useful guardrails

Choose your microphone, skip silent recordings, set duration limits, and optionally mute built-in speaker audio while recording.

Accuracy

Custom vocabulary

Bias the model toward names, acronyms, product terms, and phrases that generic dictation often gets wrong.

Cleanup

Text replacements

Fix predictable mistakes automatically, such as replacing "Cloud Code" with "Claude Code" after transcription.

Recovery

History and retry

Copy recent transcriptions, inspect local debug sessions, play back captured audio, and retry failed transcriptions.

Engines

Choose local by default. Use cloud only when you want it.

Overwhisper supports multiple transcription paths because speech quality, languages, and latency needs vary. You stay in control.

Privacy

Your voice should not need a round trip to a server.

Local transcription keeps the default path simple: audio is recorded, transcribed, and inserted on your Mac. Cloud transcription is opt-in.

Recent debug sessions are stored locally so you can retry failures and troubleshoot. They are capped and clearable from Settings.

Local models by default

WhisperKit and Parakeet process speech on Apple Silicon without sending audio to a transcription service.

Optional cloud path

OpenAI is used only when selected, or when you enable cloud fallback and local transcription fails.

No app analytics

Overwhisper does not include app telemetry or behavior tracking. API keys are stored in macOS Keychain.

Workflow

From thought to text in one gesture.

Focus a field

Put your cursor in any app where you want the text to appear.

Press a hotkey

Toggle recording or hold push-to-talk, depending on how you work.

Speak naturally

The overlay confirms recording state and input level while you dictate.

Keep working

Overwhisper transcribes, cleans up replacements, and pastes the result.
